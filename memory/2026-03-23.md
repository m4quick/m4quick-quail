## Quail Hatch Video Project

Working on a viral-worthy quail hatching video for YouTube Shorts/TikTok.

### Video Iterations
- **v1/v2**: Initial attempts with synthetic audio
- **v3**: Added real ukulele/acoustic background music downloaded from YouTube Audio Library
- **v4**: Fixed text layout - split long captions onto multiple lines and adjusted positioning so text doesn't run off screen. Smaller font sizes for better readability. Centered text with black stroke outline for contrast.

### Technical Approach
- Using MoviePy 1.0.3 with PIL for text rendering (avoiding ffmpeg drawtext filter issues)
- 9:16 vertical format (1080x1920), 30 seconds, 30fps
- H.264/AAC encoding at 6M bitrate
- Text: White fill with black stroke (stroke_width=4) on transparent PNG overlays
- Multi-line captions to fit narrow vertical format
